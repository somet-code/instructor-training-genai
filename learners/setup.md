---
title: Setup
---

No specialist software is required to participate in this bonus module.
You will need a modern web browser.

If you have an account for a chatbot application (e.g. ChatGPT, Claude, Gemini, Mistral Vibe), you can use that during the training.
If you do not already have an account, there is no need to create one unless you want to.
Participants who prefer not to create an account can use the [duck.ai][duck-ai] chatbot, which does not require a login.
